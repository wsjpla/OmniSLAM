# Introduuction
Manually
Compile Opencv
```
cmake -D CMAKE_BUILD_TYPE=RELEASE -D WITH_CUDA=ON -D OPENCV_EXTRA_MODULES_PATH=../../opencv_contrib-3.4.8/modules/ -D CMAKE_BUILD_TYPE=RELEASE  -D ENABLE_FAST_MATH=1  -D CUDA_FAST_MATH=1 -D WITH_CUBLAS=1  -DENABLE_PRECOMPILED_HEADERS=OFF -DBUILD_opencv_cudacodec=OFF  -DCUDA_TOOLKIT_ROOT_DIR=/usr/local/cuda/  -D BUILD_EXAMPLES=OFF  -D BUILD_DOCS=OFF -D BUILD_PERF_TESTS=OFF -D BUILD_TESTS=OFF WITH_TBB=ON -D WITH_OPENMP=ON -D WITH_IPP=ON  -D BUILD_opencv_apps=OFF -DCUDA_ARCH_BIN=62 ..
```
